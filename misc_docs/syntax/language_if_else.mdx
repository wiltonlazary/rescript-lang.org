Use `if / else` expressions to express a value through a `true` / `false` condition.

### Example

<CodeTab labels={["ReScript", "JS Output"]}>

```res
let user = "Anna"

let greeting = if user === "Anna" {
  "Hi Anna!"
} else {
  "Hi unknown!"
}
```

```js
var user = "Anna";

var greeting = user === "Anna" ? "Hi Anna!" : "Hi unknown!";
```

</CodeTab>
