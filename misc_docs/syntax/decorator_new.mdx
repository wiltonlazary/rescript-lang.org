The `@new` decorator is used whenever you need to bind to a JavaScript
class constructor that requires the `new` keword for instantiation.

### Example

<CodeTab labels={["ReScript", "JS Output"]}>

```res
type t

@new external create: unit => t = "Date"

let now = create()
```

```js
var now = new Date();
```

</CodeTab>
