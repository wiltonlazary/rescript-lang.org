This operator performs *floating point* subtraction.

<CodeTab labels={["ReScript", "JS Output"]}>

```res
let result = 3.0 -. 2.5
```

```js
var result = 3.0 - 2.5;
```

</CodeTab>

For subtracting *integers* see the `-` operator.
