A `char` literal is composed of two **single** quotes. Double quotes are reserved for the `string` type. Note that `char` doesn't support Unicode or UTF-8.

### Example

<CodeTab labels={["ReScript", "JS Output"]}>

```res
let one = '1';
let upperCaseA = 'A';
let lowerCaseA = 'a';
```

```js
var one = /* '1' */49;

var upperCaseA = /* 'A' */65;

var lowerCaseA = /* 'a' */97;
```

</CodeTab>
